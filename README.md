My tool box in Python.

Functions are grouped into:

- Health
  - Data summary: `health.clinical`
  - eCTD: `health.ectd` 
  - MedDRA: `health.emt` 
- General 
  - Data summary: `util.cdt`
  - Graphs: `util.cdtg`
  - System: `util.lsr`
- Standard: 
  - CDISC 
    - Library: `stdcd.api`
  - WHO
    - B3/C3
- StatLab: 
  - Reliability: `statlab.kappa`
  - Sampling

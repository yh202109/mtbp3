   .. py:method:: count_to_cate(group_columns, target_column, bins, stats)

      Convert numerical data into categorical data based on specified bins and calculate statistics.

      :param group_columns: The columns to group by.
      :type group_columns: list
      :param target_column: The column to convert and calculate statistics on.
      :type target_column: str
      :param bins: The bin edges for converting the target_column.
      :type bins: int or list or numpy.ndarray
      :param stats: The statistics to calculate.
      :type stats: list

      :returns: The resulting DataFrame with calculated statistics.
      :rtype: out1 (pandas.DataFrame)

      :raises ValueError: If the input is not a DataFrame or group_columns is not a list or contains non-string elements,
          target_column is not a non-empty string or does not exist in the DataFrame,
          target_column is not a numerical column, bins is not an integer or a sequence of bin edges,
          or stats is not a list of strings or contains invalid statistics.

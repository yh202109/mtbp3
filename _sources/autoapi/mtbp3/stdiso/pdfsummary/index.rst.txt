   .. py:method:: get_image(page_index=0, image_index=0, outfolder='')

      Retrieves an image from the PDF summary.
      :param page_index: The index of the page containing the image. Defaults to 0.
      :type page_index: int, optional
      :param image_index: The index of the image within the page. Defaults to 0.
      :type image_index: int, optional
      :param outfolder: The folder to save the image file. Defaults to "".
      :type outfolder: str, optional

      :returns: If `outfolder` is provided, the function saves the image file and returns the file path as a string.
                Otherwise, it returns the image as a PIL.Image object.
      :rtype: str or PIL.Image

      :raises ValueError: If `page_index` or `image_index` is not a non-negative integer, or if they are out of range.

   .. py:method:: get_outline_list(max_itr=5)

      Returns the outline structure of the PDF file.

      :param max_itr: The maximum number of iterations to traverse the outline structure. Defaults to 5.
      :type max_itr: int, optional

      :returns: A list containing the outline structure and any unprocessed elements.
      :rtype: list

      :raises ValueError: If max_itr is not an integer between 1 and 20.

      .. note::

         The outline structure is obtained from the `outline` attribute of the `pp` object.
         The returned list contains the outline structure in a hierarchical format, along with any unprocessed elements.



   .. py:method:: show_outline_tree(max_itr=5, to_right=False)

      :param to_right: A flag indicating whether the tree should be displayed to the right. Defaults to False.
      :type to_right: bool, optional

      :returns: A formatted string representation of the outline structure.
      :rtype: str

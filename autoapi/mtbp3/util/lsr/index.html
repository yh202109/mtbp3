list_files()[source]¶

List files in the specified directory based on the output format.

Returns:

The list of files, JSON string, or DataFrame based on the output format.

Return type:

list or str or pd.DataFrame

Examples

>>> lsr = LsrTree("/path/to/directory", outfmt="list")
>>> lsr.list_files()
['file1.txt', 'file2.txt', 'file3.txt']

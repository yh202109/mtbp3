get_outline_list(max_itr=5)[source]

Returns the outline structure of the PDF file.

Parameters:

max_itr (int, optional) – The maximum number of iterations to traverse the outline structure. Defaults to 5.

Returns:

A list containing the outline structure and any unprocessed elements.

Return type:

list

Raises:

ValueError – If max_itr is not an integer between 1 and 20.

Note

The outline structure is obtained from the outline attribute of the pp object. The returned list contains the outline structure in a hierarchical format, along with any unprocessed elements.
